# 🚀 Featured Repositories & Security Labs

### 🔐 [DevSecOps Pipeline Lab](https://github.com/delgadoroberto/devsecops-pipeline-lab)
Hands-on project integrating CI/CD security validation, dependency scanning, container security, infrastructure-as-code security testing, and automated security checks using tools like **Trivy** and **OWASP ZAP**.

### ☁️ [Terraform AWS Security Labs](https://github.com/delgadoroberto/terraform-aws-security-labs)
Security-focused Terraform labs implementing secure AWS configurations, IAM security controls, encryption policies, and automated infrastructure hardening practices.

### 🛡️ [Linux Security Audit](https://github.com/delgadoroberto/linux-security-audit)
Automated shell scripting labs focused on parsing system configurations, validating security postures against standard baselines (CIS Benchmarks), and gathering system audit logs.

### 📊 [Log Analyzer for Security Monitoring](https://github.com/delgadoroberto/log-analyzer)
A Python-based log analysis utility designed to parse web/system server logs, detect anomalous traffic patterns, and flag potential indicators of compromise (IoCs).

# ⚙️ Profile GitOps & Security

This profile repository applies DevSecOps principles to its own deployment. Every commit triggers an automated CI/CD pipeline via GitHub Actions (`readme-check.yml`) that enforces:
- **Markdown Integrity Validation:** Ensures the structural correctness and availability of key profile documentation.
- **Automated Secret Scanning:** A localized linter process designed to catch and prevent accidental exposures of plain-text credentials or API keys prior to merging.
